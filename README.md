# FluentUIBlazorTestAppWebAssembly

This branch of the FluentUIBlazorTestApp demonstrates the behavior of the Autocomplete component when the SelectedOption is set from the outside.

It is used as sample repo for the issue https://github.com/microsoft/fluentui-blazor/issues/4380.

* The Home page demonstrates and explains the behavior in general.
* The *Grid Form Example* describes a use-case where the SelectedOption is set from the outside, where the described behavior **does not** cause any issues.
* The *Dependent Inputs Example* describes a use-case where the SelectedOption is set from the outside, where the described behavior **does** cause an issues.
